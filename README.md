# Prakrati — Leads CRM

Private, self-contained lead-management web app for Sanaya Real Estate, Dubai (agent: Prakrati).

- Single static `index.html` (no server, no build) — 3,523 leads with country detection + WhatsApp outreach.
- All edits save locally in the user's browser (localStorage).
- Access is protected by a team password gate.
